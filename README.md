# Exodus LARP Wallet

**Exodus LARP Wallet** is a fictional cryptocurrency wallet created for live-action role-playing (LARP) games. It is designed for immersive roleplay and **does not store, send, or receive real cryptocurrency**.

## Features

* Fictional Bitcoin (BTC) and Solana (SOL) balances.
* Players can choose any in-game balance for their character.
* **Ctrl+A** opens the send dialog, where the player selects how much fictional currency to send.
* Pressing **Ctrl+A** again confirms and completes the fictional transaction.
* Simulated transaction history.
* Designed for roleplay events, storytelling, and demonstrations.
* No blockchain connection.
* No real funds.
* No cryptocurrency keys or wallets.

## Disclaimer

This project is intended **for entertainment and LARP purposes only**. All balances, transactions, and cryptocurrencies shown are fictional and have no monetary value. It is not affiliated with or endorsed by the Exodus cryptocurrency wallet or any blockchain network.
